 🎓 Student Database Management System

A command-line based Student Database Management System developed in C for efficient management of student records. The application provides a menu-driven interface to perform various operations such as adding, viewing, searching, modifying, sorting, and deleting student information.

This project demonstrates core concepts of C programming including:

- Structures
- Dynamic Memory Allocation
- Linked Lists
- File Handling
- Modular Programming
- Data Persistence
- Sorting & Searching Algorithms

---
 📖 Project Overview

Managing student records manually can be time-consuming and error-prone. This project provides a simple and efficient solution for storing and managing student information through a console-based application.

The system allows administrators to:

- Add new student records
- Display all student records
- Search student information
- Modify existing records
- Delete student records
- Sort records using different criteria
- Save and retrieve data from files
